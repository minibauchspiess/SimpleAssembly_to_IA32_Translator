Para compilar: insira o comando 'make' no terminal. É utilizado um arquivo Makefile para a compilação (incluso nos arquivos enviados).

O executável se chamará 'tradutor'. Para traduzir um programa, digite './tradutor <nome_do_arquivo>', a saída será um arquivo .s

Na pasta 'IA32_Sections' estão as inicializações das sessões, incluindo as funções requisitadas em 'Text'

Obs: Os arquivos são gerados na mesma pasta em que o arquivo de entrada se encontra
